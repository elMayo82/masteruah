## Datos

### Financieros
Los datos financieros se *limitan* a expresar cifras **económicas**

- Cobros
- Pagos

- Ingresos
- Gastos

- Beneficios
- Costes

### Logísticos
Los datos ~~financieros~~ logísticos expresan acciones y cifras relacionadas con los **bienes** y **servicios** que se ofrecen 

- Compras
- Ventas
